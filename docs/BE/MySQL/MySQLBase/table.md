## 字段

PK：primary key 主键

NN：not null 非空

UQ：unique 唯一索引

BIN：binary 二进制数据(比text更大的二进制数据)

UN：unsigned 无符号     整数（非负数）

ZF：zero fill 填充0 例如字段内容是1 int(4), 则内容显示为0001 

AI：auto increment 自增

G：generated column 生成列

在mysql查询语句中加\g、\G的意思：

\g  的作用是分号和在sql语句中写“；”是等效的
\G  的作用是将查到的结构旋转90度变成纵向（换行打印）
